# 🎵 Music Finder Pro v27

A powerful YouTube music discovery tool with precise time filtering, genre selection, and high-quality MP3 downloader.

## ✨ Features
- **Precise Time Filtering**: Search from last 24h up to 3 years ago.
- **Genre-Based**: Optimized keywords for various global and local genres.
- **Duration Control**: Filter between single tracks or full albums.
- **Modern UI**: Dark mode, vibrant orange & blue accents.

## 🚀 How to Run (Desktop)
1. Clone this repo: `git clone https://github.com/username/music-finder-pro.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python app_desktop.py`

## 🌐 Streamlit Version
You can also run the web version using:
`streamlit run app_web.py`
